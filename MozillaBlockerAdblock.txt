! Title: Mozilla Blocker Adblock
! Description: This blocklist aims to block all traffic to "Mozilla: Not a Browser/Tech Company - A Political, AI, Data Collection and Advertisement Organization".
!              Mozilla accounts are blocked, mozilla download cdn and archive is blocked. Firefox forked browser's auto update attempts will be blocked.
!              Just avoid official firefox altogether and use other forks like Librewolf, Floorp, Zen Browser etc.
!              Use a package manager or download manualy to update firefox forked browsers instead of using the autoupdate feature inside the browser itself.
!              Only addons urls are not blocked so that extensions can be installed from Firefox Add-ons.
!              Use in a systemwide Adblocker like Zen systemwide adblocker from github or Adguard which is available for Android, iOS, Windows, and Mac. 
!              Using this blocklist in an adblocker inside mozilla firefox browser won't prevent mozilla itself from collecting data from and running experiments on its users.
!              Zen system wide Blocker(https://github.com/anfragment/zen) can detect and block google trackers when you visit mozilla website and your browser updates add-ons on the backgruond, Adguard for windows can't. 
!              Not Even Ublock Origin is allowed to block google trackers on mozilla sites in any firefox forked browser.
!              This is due to Mozilla's policy forced onto extension developers. Which should tell you Mozilla's privacy stance as an organization.
!              Among Firefox forks, only Palemoon and Basilisk can detect and block google trackers on mozilla websites.
